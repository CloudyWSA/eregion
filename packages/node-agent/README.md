# @eregion/node-agent

A programmatic OpenTelemetry agent for Node/Bun backends: reports handler + SQL to the daemon so the AI can answer "what query is behind this?".

Part of [**eregion**](https://github.com/CloudyWSA/eregion) — select a component in your running app, describe the change, and an AI edits the real source.

```bash
npm create eregion@latest
```

The scaffolder detects your framework and installs only the packages you need. See the [monorepo README](https://github.com/CloudyWSA/eregion#readme) for the full picture.
